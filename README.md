# Funcións lóxicas V 0.00

## version 01.

Para comprobar os resultados dos exercicios teóricos en clase.
usar un multiplexor (Mux), simulando unha pequena PLD de catro entradas e 2^4 = 16
 operacións posibles. 
 Utiliza as entradas de control como entrada de datos A,B,C,D, 
 e as entradas I0..I15 como selección das funcións lóxicas minitérmos.
